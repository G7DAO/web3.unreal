External RPC
=======================

This endpoint is used to call external rpc providers using the standard json-rpc format.

JSON-RPC Documentation
----------------------

Ethereum: https://ethereum.org/en/developers/docs/apis/json-rpc/

MetaMask: https://docs.metamask.io/guide/rpc-api.html#table-of-contents

Polygon: https://wiki.polygon.technology/docs/edge/get-started/json-rpc-commands/

For other EVM chains, check with their developer docs.
